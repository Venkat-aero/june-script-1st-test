Function Declaration

  • We can call the function before declaring it .
  • SYNTAX
  • function function name (parameters) - while declaring it

    function name (Arguments) - while calling it
  • Ex:-

    sum(3,4);

    console.log(result);

    function sum(a,b){

    return a+b;}

    console.log(result);
